# 최종 프로젝트 명세서: 밸런스게임 "NEON DIVE"

## 1. 서비스 개요

- **콘셉트:** 재미를 뽑는 네온 자판기
- **한 줄 설명:** 술자리의 분위기를 고조시키는 가장 빠른 방법, 너의 선택은?
- **목적:** 20대 초중반을 타겟으로, 술자리나 파티에서 즉시 사용할 수 있는 인터랙티브 밸런스게임 웹사이트 제공. 로그인 없이 빠른 플레이와 세련된 시각적 경험에 집중한다.

## 2. 디자인 및 UX 원칙

- **디자인 콘셉트: 'NEON DIVE'**
  - 어두운 라운지 바의 분위기를 연출하되, 네온사인의 강렬한 포인트 컬러로 사용자의 시선과 행동을 유도합니다.
  - Glassmorphism(유리 효과), 미세한 애니메이션, 동적인 화면 전환을 통해 "센스 있는 서비스"의 인상을 줍니다.

- **핵심 UX 원칙**
  - **Dynamic First View:** 사용자가 접속 즉시 정적인 화면이 아닌, 살아 움직이는 듯한 인터랙티브 화면을 마주하여 흥미를 유발합니다.
  - **Desktop-First Layout:** 데스크탑(웹) 환경에 우선적으로 최적화된 넓은 가로형 레이아웃을 제공합니다.
  - **Intuitive Flow:** `메인 화면` → `카테고리 선택` → `게임 시작`으로 이어지는 흐름을 명확하고 단순하게 유지합니다.
  - **Rapid Gameplay:** 선택지가 2개인 질문의 경우, 좌/우 병렬 배치하여 빠른 비교와 선택을 유도하며, 선택 후 불필요한 대기 시간 없이 다음 질문으로 신속하게 전환됩니다.

## 3. 최종 구현 기능

### Front-end
- **메인 시작 화면:**
  - '네온 자판기' 콘셉트의 Glassmorphism(유리 효과) 패널 UI.
  - 패널 내부에서 카테고리 아이콘(😋, 🔥, 😈, 🤪)들이 부드럽게 떠다니는 동적 애니메이션 구현.
  - 중앙에 `🎲 바로 시작` CTA 버튼을 배치하여 사용자의 행동을 유도.

- **카테고리 선택 화면:**
  - '바로 시작' 버튼 클릭 시, 떠다니던 아이콘들이 확장되며 카테고리 선택 버튼으로 전환되는 애니메이션.
  - 데스크탑에 최적화된 2x2 그리드 레이아웃으로 4개의 카테고리(`가벼운 맛`, `논쟁 유발`, `위험한 맛`, `아무거나!`)를 제공.
  - 각 버튼은 아이콘과 텍스트로 구성되며, Hover 시 네온광 효과가 강조됨.

- **게임 화면 (`game-screen` Web Component):**
  - 질문 텍스트와 2개의 선택지 카드가 좌/우로 병렬 배치된 데스크탑 우선 레이아웃.
  - 각 선택지 카드는 거대한 터치 영역을 제공.
  - 사용자가 선택한 카드에만 네온광(`box-shadow`) 효과를 주어 시각적 피드백을 제공.
  - **(기획 변경)** 초기 프로토타입의 퍼센트 결과 표시는 빠른 게임 진행을 위해 최종 버전에서 제거됨. 선택 후 1초 뒤 다음 질문으로 자동 전환.

- **라우팅 및 공유:**
  - 질문별 고유 URL(`/q/:id`)을 통해 특정 질문으로 직접 접근 및 공유 가능.
  - 브라우저의 뒤로가기/앞으로가기 기능 지원.

### 데이터
- **로컬 데이터:** 전체 질문 데이터셋(50개)은 서버 연동 없이 `data.js` 파일 내에 JSON 배열 형태로 저장되어 있음.
- **상태:** 현재 버전은 실제 사용자 데이터와 연동되지 않는 **Front-end Only 프로토타입**임.

## 4. 기술 스택

- **Core:** Vanilla HTML, CSS, JavaScript (ES Modules)
- **UI:** Web Components를 활용하여 `game-screen` 등 핵심 UI를 컴포넌트화.
- **Styling:** CSS Variables를 통한 체계적인 컬러 시스템 관리 및 Keyframe 애니메이션.
